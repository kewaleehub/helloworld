## 🚀 Run the Project

### 1. Clone this repository

```bash
git clone https://github.com/your-username/noi-ascii-garden.git
```

### 2. Enter the project folder

```bash
cd noi-ascii-garden
```

### 3. Run the Python file

```bash
python garden.py
```
